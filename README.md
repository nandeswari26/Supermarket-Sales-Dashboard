# 🛒 Supermarket Sales Dashboard (Power BI)

## 📊 Overview

This project is a Power BI dashboard created to analyze supermarket sales data.

## 📌 Features

* Total Sales, Profit, and Orders KPIs
* Sales by Category, Product, and Region
* Monthly Sales Trend
* Interactive slicers (Month, Category, Product, Region)

## 🛠 Tools Used

* Power BI
* Excel
